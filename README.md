# Orbital Simulator

This was my A Level Computer Science Coursework, that aimed to bring the excellent intuition that games like Kerbal Space Program provide about orbital mechanics into an educational setting.

## Features

The coursework features fully simulated n-body orbital physics, and also calculates a planet's two-body orbital charactersitics to display. These can also be modified in real time, for example changing the eccentricity of a planet's orbit by moving a slider. It also features a fully custom camera engine to lock onto and pivot around the planets.
